Overview

simple_net is a production-ready TCP socket abstraction library for Eiffel. It provides intuitive, Design-by-Contract-based APIs for building networked applications with client-server communication.

Unlike raw socket APIs that require extensive boilerplate, simple_net reduces typical TCP client/server implementations from 50-100 lines of code to just 5-10 lines, while maintaining full control and adding automatic error handling through contracts.

Part of the Simple Eiffel ecosystem.

Key Features

Intuitive Client API

TCP client with simple connect, send, receive, close operations. No boilerplate.

Server Socket

TCP server with listen, accept, and configurable backlog for high-throughput applications.

Address Validation

Immutable network endpoint (host:port) with comprehensive IPv4 validation.

Semantic Errors

ERROR_TYPE classification instead of raw codes: connection_refused, timeout, etc.

Design by Contract

Full preconditions, postconditions, and invariants on all features.

SCOOP Compatible

Thread-safe design with proper separation semantics for concurrent programs.

Quick Start

See Quick API Reference for immediate examples.

Basic Client Example

-- Create client and connect
client := create {SIMPLE_NET}.new_client_for_host_port ("example.com", 8080)
if client.connect then
    client.send_string ("Hello, server!")
    message := client.receive_string (1024)
    client.close
end

Basic Server Example

-- Create and listen
server := create {SIMPLE_NET}.new_server_for_port (8080)
if server.listen (10) then
    connection := server.accept
    if connection /= Void then
        -- Process connection
    end
    server.close
end

For more examples, see the User Guide and Cookbook.

Installation

Add simple_net to your Eiffel project by including it in your ECF configuration file:

<library name="simple_net" location="$SIMPLE_EIFFEL/simple_net/simple_net.ecf"/>

Requires Eiffel 25.02 or later with ISE EiffelStudio base and net libraries.
